# SpO2 Measurement PCB

## Overview
This project is a custom PCB designed in KiCad for an SpO2 (blood oxygen saturation) monitoring circuit.

## Features
- Photodiode sensing circuit
- LED driver for IR and red LEDs
- Signal conditioning stage
- Microcontroller interface

## Tools Used
- KiCad 9
- GitHub for version control

## Files Included
- Schematic
- PCB layout
- Gerber files
- Custom footprint libraries
